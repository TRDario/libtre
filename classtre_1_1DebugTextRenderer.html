Detailed Description

Debug text renderer.

The renderer is conceptualized around two independent "pens", a left-aligned and right-aligned one, writing text sequentially. Writing uses special text formatting, the specifics of which you can see at Formatted Debug Text Format.

Only one instance of the debug text renderer is allowed to exist at a time.
