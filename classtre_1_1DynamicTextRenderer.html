Detailed Description

Renderer for one-off or frequently changing text.

Implemented as an abstraction layer that forwards its output to the 2D renderer.

Parameters
priorityThe drawing priority of the text (higher is drawn on top).
textThe text string. See Formatted Rendered Text Format for the specifics of the text format.
fontThe font to use for the text.
fontSizeThe font size to use for the text.
textColorsThe available text colors. By default, the color at index 0 is used. Must not be empty, otherwise a failed assertion may be triggered.
outlineThe outline parameters to use for the text, or NO_OUTLINE.
textboxThe textbox to frame the text around.
