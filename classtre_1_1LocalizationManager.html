tre::LocalizationManager::LocalizationManager ( const std::filesystem::path & file)

Loads a localization manager.

For specifics of the localization file format see Localization File Format.

Exceptions
tr::FileNotFoundIf the file is not found.
tr::FileOpenErrorIf opening the file fails.
std::bad_allocIf any internal allocations fail.
LocFileParseWithErrors

If any parsing errors occur during the parsing.

Parsing errors are collected and recovered from during the loading process, and a localization manager in a valid state is produced even if errors are thrown.

You may extract the manager and the error information from the thrown exception.

Parameters
[in]fileThe path to the localization file.
