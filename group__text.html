Detailed Description

Text rendering functionality.

An instance of Renderer2D must be created before any renderers from this section except DebugTextRenderer can be used.

tr::TTFont & font,

Renders text to a bitmap according to the format described in Formatted Rendered Text Format.

Exceptions
std::bad_allocIf an internal allocation occurs.
tr::BitmapBadAllocIf allocating a bitmap fails.
Parameters
[in]textThe text to render. Must not be an empty string or one solely consisting of control sequences.
[in]fontThe font to use in rendering.
[in]sizeThe size of the font.
[in]dpiThe dpi to use for the text.
[in]maxWidthThe width limit of the resulting bitmap.
[in]alignmentThe alignment of the text within the bitmap.
[in]textColorsThe colors to use when rendering the text. The span cannot be empty. By default, the color at index 0 is used.
[in]outlineThe outline parameters to use for the text, or NO_OUTLINE.
Returns
A bitmap containing the text.
