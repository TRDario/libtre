Detailed Description

Provides utilities for basic renderer checks.

libtre prefers to work with self-contained renderers, as opposed to directly manipulating OpenGL state. This can cause inefficiencies if repeated, redundant state changes are made. This file thus provides helper functions to minimize switching state when not needed.
