Formatted Rendered Text Format

text_renderering.hpp features an option to render formatted, multistyle text. The following formatting escape sequences are supported for manipulation of the text style:

\\

  • Escape sequence for outputting '\'.

\c[i] (where i is 0-9)

  • Sets the text color to color at index n of the span of colors passed to the function. If i is an invalid index, nothing happens.

\!

  • Resets the text color to the default. For the bitmap text manager, this means a completely white tint, while on the other renderers it's equivalent to \c0.

\b

  • Toggles whether the text is drawn bold (not available for the bitmap text manager).

\i

  • Toggles whether the text is drawn italic.

\u

  • Toggles whether the text is drawn underlined (not available for the bitmap text manager).

\s

  • Toggles whether the text is drawn striked out (not available for the bitmap text manager).

Unknown escape sequences are ignored.
