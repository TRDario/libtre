Formatted Rendered Text Format

text_renderer.hpp features an option to render formatted, multistyle text. The following formatting escape sequences are supported for manipulation of the text style:

\\

  • Escape sequence for outputting '\'.

\c[i] (where i is 0-9)

  • Sets the text color to color at index n of the span of colors passed to the function. If i is an invalid index, nothing happens.

\b

  • Toggles whether the text is drawn bold.

\i

  • Toggles whether the text is drawn italic.

\u

  • Toggles whether the text is drawn underlined.

\s

  • Toggles whether the text is drawn striked out.

Unknown escape sequences are ignored.
